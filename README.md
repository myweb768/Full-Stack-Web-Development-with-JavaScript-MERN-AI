# Full-Stack-Web-Development-with-JavaScript-MERN-AI

Module 4 Assignment

Welcome to the third assignment of our course. In this assignment, you have to create a Responsive Premium Portfolio Website using Responsive Design + Grid + Flexbox + Tailwind CSS / Raw CSS.

Assignment Instructions:

1. Create Main Layout (Grid System)
   ● Create a parent container using CSS Grid
   ● Show 3 sections in one row for desktop
   ● Show 2 sections in tablet
   ● Show 1 section in mobile

2. Create Navbar (Flexbox)
   Navbar should contain:
   ● Logo / Name
   ● Home
   ● About Me
   ● Portfolio
   ● Services
   ● Contact
   Use Flexbox for alignment.

3. Create Hero Section
   Hero section should contain:
   ● Profile Image
   ● Heading (Your Name)
   ● Short Intro Text
   ● Button (Hire Me / Contact Me)
   Use Flexbox and make it responsive.

4. Create Portfolio Section
   Create 6 project cards.
   Each card should contain:
   ● Project Image
   ● Project Title
   ● Short Description
   ● View Button
   Use Grid Layout. Make it responsive.
