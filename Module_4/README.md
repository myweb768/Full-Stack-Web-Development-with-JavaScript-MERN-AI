# Full-Stack Web Development with JavaScript (MERN + AI)

## Module 4 Assignment: Responsive Premium Portfolio

Welcome to the fourth module assignment! In this project, the goal is to build a high-quality, responsive portfolio website leveraging modern layout techniques like **CSS Grid**, **Flexbox**, and **Tailwind CSS**.

---

## 🚀 Project Overview

The objective is to create a seamless user experience across all devices (Desktop, Tablet, and Mobile) using a mobile-first or responsive-first approach.

### 🛠 Tech Stack Used

- **HTML5:** Semantic structure.
- **CSS3 / Tailwind CSS:** Advanced styling and layout.
- **JavaScript:** Interactivity.
- **Font Awesome:** Iconography.

---

## 📋 Assignment Instructions

### 1. Main Layout (Grid System)

Implement a robust parent container using **CSS Grid** to handle the overall page structure:

- **Desktop:** 3 sections per row.
- **Tablet:** 2 sections per row.
- **Mobile:** 1 section per row (stacked).

### 2. Navigation Bar (Flexbox)

Design a functional navbar using **Flexbox** for alignment. It must include:

- Logo / Brand Name
- Home
- About Me
- Portfolio
- Services
- Contact

### 3. Hero Section

The "above-the-fold" area should be engaging and responsive, containing:

- **Profile Image:** High-quality professional photo.
- **Heading:** Your Name.
- **Intro Text:** A brief, catchy professional summary.
- **CTA Button:** "Hire Me" or "Contact Me."

### 4. Portfolio Section

Showcase your work using a **Grid Layout** featuring 6 project cards. Each card must include:

- Project Thumbnail Image.
- Project Title.
- Short Description.
- "View Project" Button.
